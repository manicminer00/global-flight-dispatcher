What does Reset Custom Database do?
Step-by-Step Reset Process:
  • Confirmation: A dialog asks for permission to wipe custom data.
  • Targeted Deletion: Only the specific keys (dispatcher_custom_user_airports, dispatcher_custom_airports, and dispatcher_custom_fleet) are removed from your browser's local memory.
  • Scope: This does not clear cookies, bookmarks, or browsing history; it only affects this specific dispatcher tool.
  • Refresh: The interface instantly updates to reflect the empty database.
